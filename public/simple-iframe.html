How to Use URL Parameters

You can filter the map by adding parameters to the iframe URL:

  • ?what=TERM - Filter by search term
  • ?where=LOCATION - Filter by location
  • ?category=CATEGORY - Filter by category (museum, cultural, landmark, transportation, education, park)

Parameters can be combined with & (e.g., ?what=museum&category=cultural)
